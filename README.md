# bincom_interview

### Install Virtual environment
##### For Windows
-       pip install virtualenv
##### For Linux or IOS
-       pip3 install virtualenv

### Create Virtual environment
##### For Windows
-       python -m virtualenv <any-name-of-choice>
##### For Linux or IOS
-       python3 -m virtualenv <any-name-of-choice>

### Install Django v4.0
-       pip install django==4.0
